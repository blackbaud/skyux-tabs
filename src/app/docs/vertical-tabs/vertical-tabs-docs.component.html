Vertical tabs divide and categorize page content when the number of categories is large.

Use vertical tabs when the number of tabs and/or the tab label text is expected to exceed the horizontal space available.

Use grouped vertical tabs when the tabs naturally fit into multiple categories.

Don’t use vertical tabs when horizontal tabs are also suitable.

Don’t combine vertical tabs and horizontal tabs. Change the workflow instead of nesting multiple sets of tabs.

Don’t use vertical tabs when the user’s workflow will require them to view and perform actions on each section. Use a split view instead.

Don’t use vertical tabs inside a modal. Use a sectioned form instead. Although the two look similar, sectioned form works better with modal interactions.
